Image Classification using ResNet50

This project implements an image classification model using ResNet50 and Transfer Learning.

The notebook includes:

Data preprocessing and augmentation

Training a pretrained ResNet50 model

Model evaluation

Accuracy and loss visualization

Files

ResNet50_Final.ipynb: main notebook for training and testing the model

Requirements

Python libraries used:

numpy

pandas

matplotlib

tensorflow / keras

torch / torchvision

scikit-learn

opencv-python

How to Run

Open the notebook and run all cells in order.

Model

Base model: ResNet50 (pretrained on ImageNet)

Optimizer: Adam

Loss function: Categorical Crossentropy

Metric: Accuracy
